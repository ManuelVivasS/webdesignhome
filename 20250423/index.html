DEFINICION

El HTML semántico es el uso de las etiquetas HTML para describir el significado del contenido de una página web, en lugar de solo cómo se presenta. Esto ayuda tanto a los humanos como a las máquinas a comprender la estructura y el propósito de una página web.

Es una practica recomendable usar elementos semánticos porque otorgan significado

BENEFICIOS

Los beneficios de usar etiquetas semaánticas incluyen:

  • Optimización SEO:Facilitan la indexación y mejoran la visibilidad en motores de búsqueda.
  • Estilos consistentes:Ayudan a aplicar estilos CSS adecuados basados en la estructura semántica del contenido.
  • Mejora la accesibilidad: Ayudan a los usuarios con discapacidades a comprender mejor el contenido de la página.

ELEMENTOS

Los elementos semánticos en HTML son etiquetas que describen el significado de su contenido de manera clara. Estos elementos mejoran la estructura del código y facilitan la comprensión tanto para los desarrolladores como para los navegadores y motores de búsqueda. Aquí tienes algunos ejemplos de elementos semánticos:

  • <header>Representa la cabecera de una página o sección.
  • <nav>Define una sección para enlaces de navegación.
  • <main>Contiene el contenido principal de una página.
  • <aside>Utilizado para contenido adicional, como barras laterales o notas.
  • <footer>Representa el pie de página de un documento o sección.
  • <section>Agrupa contenido relacionado.
  • <articule>Indica contenido independiente, como artículos o noticias.
  • <figure>Agrupa contenido como imagenes y sus descripciones
  • <>
